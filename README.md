# artfave

**ArtFaveは、ローカルの画像ファイルを簡単に閲覧するためのFlutterアプリケーションです。**

**使用するにはリリースページからexeファイルの入ったzipファイルをダウンロードするか、自分でビルドしてください。**

**使用は自己責任でお願いします。**
